설비 상태 모니터링, Trend·Spectrum·Waterfall 분석,
역할별 맞춤 화면을 제공하는 CMS-5000을 설치합니다.

⬇ 설치 프로그램 다운로드 (CMS5000Setup.exe)

설치 방법

1
CMS5000Setup.exe를 다운로드합니다.
2
파일을 더블클릭해 설치합니다.
3
바탕화면 바로가기로 CMS-5000을 실행합니다.

🔄 자동 업데이트

앱 실행 시 최신 버전이 있으면 백그라운드에서 자동 다운로드됩니다.
다음 재시작 시 새 버전이 자동 적용됩니다. 별도 작업 불필요.
